Title_[ 포트폴리오 모음 페이지 ] 단일 페이지 HTML5 / CSS3 / JavaScript(& Jquery) 메인컬러 :
2번째 작품
반응형 / 웹표준 / 웹접근성 / 크로스 브라우징 / 디자인
크롬 브라우저와 1920x1080 해상도에 최적화.
순서상으론 두번째지만 가장 중요한 페이지므로 첫번째에 두었다.
최대한 플러그인이나 라이브러리는 배제하고 만들어봤다.
디자인 작업은 최소화하여 포트폴리오 화면별 이미지정도만 캡쳐하거나 수정했고 그 외에는 죄다 코딩과 딩벳폰트 위주로 구현.
그러다 보니 사이트 디자인은 단순한 도형이나 텍스트 위주가 됐다.
keyframes와 Jquery, animation, transition을 활용한 애니메이션과 svg를 활용한 벡터그리기 효과 등등, 직접 코드를 칠 수 있는 것들로만 꾸며봤다.
이 쯤부터 웹호환성(크로스브라우징), 웹표준, 웹접근성 등을 위시하여 CSS 네이밍 방법론BEM,SMACSS,OOCSS 등에 관심을 가졌지만 아직까지는 그냥 개념정도로만 잡고 작업물에 적용하진 않았다.
어차피 클래스 네이밍같은 것은 회사자체 규정이 따로 있는 경우가 많다고 하기도 하고. 아직은 그냥 '이런것이 있다'정도만 알아도 되겠다 싶어서 보류.

다만 이 중에서도 웹접근성은 공부할 부분이 너무 많지만, 일단 키보드만으로 이용 할 수 있도록 최대한 노력해봤다.
그 외에도 인터페이스나 각종 인터랙티브적인 요소들도 일일이 만드는 것보다 부트스트랩같은 오픈소스 프레임워크를 다루는 것이 낫다는 말에 저런 툴들을 경험해 보는 것도 좋을 거 같긴 한데, 아직까지는 공부도 하면서 숙달도 할겸, 모두 하드코딩하는 것이 좋아보여서 계속 이렇게 작업할 예정.

물론 디자인적인 작업은 지양하고 추후 작업 부터는 무료배포하는 psd나 웹디자인 템플릿, 혹은 기존에 존재하는 다른 사이트의 이미지 소스만 가져와서 마크업 위주로 해봐야겠다.

Title_[ 놀이넷 ] 메인페이지 & 첫번째 서브페이지까지 HTML5 / CSS3 / JavaScript(& Jquery) 메인컬러 :
1번째 작품
반응형 / 웹표준 / 웹접근성 / 크로스 브라우징 / 디자인 / 기획
단순 예제나 메인페이지 제작만이 아니라 처음부터 끝까지스토리보드 스케치, 로고&아이콘 디자인, 플로우차트, 스타일 가이드, 등등 학원에서 배운 실무의 흐름대로 차근차근 과정을 밟아나갔다는 점에 의의를.
사실 구상은 잔뜩 해놨는데 정말 초보일적에 만들던거라 단순하고 기초적으로만 제작했다.
그래도 이 사이트를 만들면서 기획과 디자인의 중요성을 톡톡히 알았다. 사이트에 들어갈 컨텐츠를 짜내는 것과 디자인을 고민하는 게 어렵고 중요한 일이라는 걸 깨닫는 계기가 되었다.
포괄적으로 배우면서, 디자인은 어렵고 관심도 그냥저냥이지만 마크업과 코딩은 재밌었다.
물론 자바스크립트 부분이래봐야 기본적인 부분을 제외하면 간편한 라이브러리인 제이쿼리만 사용했으며, 그마저도 GNB는 죄다 :hover로 처리해서 주된 기능은 슬라이드 뿐이긴 하지만...

그래도 정말 단순하게 이동만 시키는 코딩에서 좀더 논리적으로 구성할 수 있을까 고민하는 과정에서 많이 부족하다는 걸 느낄 수 있었다. 배울 것이 너무나도 많고 멀리 있다는 생각을 가질 수 있었다.
여담으로 디자인이라든지 컨텐츠가 적어서 넣지 말까 생각했지만, 나름 첫 작업물인 만큼 기념이자 추후 발전한 작업물들과 비교할 때, 기준점으로 삼기 위해 등록했다.
